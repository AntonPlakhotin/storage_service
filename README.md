# Storage Service

## Описание

Storage Service отвечает за работу с базой данных PostgreSQL.

Все операции чтения и записи выполняются исключительно через данный сервис. Остальные микросервисы не обращаются к базе данных напрямую.

---

## Основные функции

- сохранение чатов;
- сохранение сообщений;
- сохранение промптов;
- получение истории сообщений;
- получение списка чатов;
- удаление данных;
- обновление информации.

## Основные сущности

### Chat

Хранит информацию о чатах пользователя.

### Message

Хранит историю переписки.

### Prompt

Хранит системные промпты.

### ChatPrompt

Связь между чатом и промптом.

---

## Используемые технологии

- Java 21
- Spring Boot
- Spring Data JPA
- Hibernate
- PostgreSQL

---

## Основные операции

| Операция | Описание |
|----------|----------|
| createChat | Создание чата |
| getChat | Получение чата |
| getChats | Получение списка |
| saveMessage | Сохранение сообщения |
| getMessages | Получение истории |
| deleteChat | Удаление |
| setPrompt | Закрепление промпта |

---

## База данных

Используются таблицы:

- chat
- message
